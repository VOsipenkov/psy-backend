--------------------
1) Форма авторизации - то что видит пользователь при входе
   логин пароль, отправляю на бэк, тот отвечает все хорошо и дает мне userId данного пользователя
          - кидаем запрос на post /users/login  (login password)
          - Проверка что такой пользователь есть и отдаем userId
          - в Таблице users поля id(он же userId), login, password в base64
2) Успешная авторизация
   значит в локал сторэдж бразуера сложили логин и userId
          - перекидываю на /chats/all/{userId} вызов чатов (передал userId в пути)
          - получил список чатов (chatId, title, createdAt)
          - отрисовал их слева и поставил указатель на первый сверху
3) Получаем список всех чатов для пользователя по userId (на бэк только userId)
4) Если тыкаешь на один из чатов то получаешь список сообщений в нем
5) Если тыкаешь в new то будет создан новый чат (видимо надо создать форму для ввода названия)
6) в п4 и в п5 в любом случае открывает окно сообщений
7) Туда по userId + chatId подгружаю список сообщений
8) Если тыкнуть отправить -то сообщение должно быть сохранено и потом уже пойдет запрос в оламу и прихраниться в БД
